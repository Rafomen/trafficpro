## Кейс 1: Товарка — интернет-магазин гаджетов (e-commerce)

**Ниша:** Электроника и гаджеты
**Гео:** Украина
**Площадка:** Facebook / Instagram Ads
**Бюджет:** $1,200/мес

### Задача
Привлечь покупательский трафик на интернет-магазин, снизить стоимость покупки и выйти на положительный ROI с первого месяца.

### Что сделали
- Сегментация ЦА: техно-энтузиасты 18–35, интересы — гаджеты, AliExpress, технологии
- Запуск карусельных объявлений и динамического ретаргетинга (DPA)
- A/B тестирование 12 креативов (видео-распаковки vs карточки товаров)
- Настройка Facebook Pixel + конверсионные события (ViewContent → AddToCart → Purchase)

### Результат
| Метрика | Значение |
|---------|----------|
| ROI | 340% |
| Стоимость покупки (CPA) | $4.20 |
| Количество заказов | 286 за месяц |
| CTR | 2.8% |

**Гео:** Украина, Казахстан
**Площадка:** Facebook Ads + воронка через Telegram-бот
**Бюджет:** $2,500/мес

### Задача
Генерация депозитных лидов (FD — First Deposit) для казино-партнёрки через Facebook с обходом ограничений площадки.

- Создание «безопасных» аккаунтов и лендингов (клоакинг)
- Тестирование 8 связок: лендинг → преленд → Telegram-бот → оффер
- Креативы: нативный контент «лайфстайл», розыгрыши, сторис-формат
- Оптимизация по событию «регистрация + депозит» через CAPI

### Результат
| Метрика | Значение |
|---------|----------|
| FD (First Deposit) | 187 за месяц |
| Стоимость FD | $13.40 |
| ROI (по выплате партнёрки) | 520% |
| Конверсия клик → FD | 1.9% |

**Гео:** Украина, Польша, Германия
**Площадка:** Google Ads (Search + Display) + Facebook Ads
**Бюджет:** $1,800/мес

- Google Ads: Search-кампании по ключам «VPN для Украины», «best VPN 2025», «скачать VPN»

**Гео:** Украина, США
**Площадка:** Email + Facebook Retargeting
**Бюджет:** $400/мес

Снизить процент чарджбэков (возвратов) и удержать клиентов, которые хотят отменить подписку на онлайн-сервис.

### Что сделали
- Сегментация: пользователи, открывшие страницу отмены → «в зоне риска»
- Ретаргетинг Facebook с оффером: скидка 50% на 3 месяца при сохранении подписки
- Email-цепочка из 3 писем: альтернативные тарифы → бонус → прощальный оффер
- Динамический креатив под каждый тарифный план

### Результат
